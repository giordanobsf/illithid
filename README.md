# illithid
Very simple neural network library in C++ based on Pytorch.

### Dependencies

- CMake (https://cmake.org)

- Matplot++ (https://github.com/alandefreitas/matplotplusplus)

### Installing

```
mkdir build
cd build
cmake ../
make
```

### Differentiable operations available for Value

- +, -, *, /, pow

- tanh
